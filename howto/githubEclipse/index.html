Sometimes the import from git in mac Eclipse is bug.

  1. Try the following fix.
  2. In the middle of import, we will be creating a new project.
  3. Unselect "Use default location."
  4. Browse and select the location to the downloaded git folder
  5. Next and finish the new project.
  6. Remember to import necessary jar.

This guideline is to setup GitHub to commit/pull in Eclipse.

  1. Token is required since August 13, 2021. Login Github using account & password in Eclipse is no longer supported.[1]
  2. Create a personal token follow the steps here: [2]
    Note that each token only appear once and cannot be retrieved again. Re-generation is needed to setup each time.
  3. Reset in Eclipse [3]
    1. Go to the Window -> Git Perspective -> Expand your Project -> Expand Remotes -> Expand the remote you want to save your password.
    2. Right-click on the Fetch or Push -> Select "Change Credentials..."
    3. Enter username and token as password -> Select Ok

Reference:
1. https://github.blog/2020-12-15-token-authentication-requirements-for-git-operations/
2. https://docs.github.com/en/github/authenticating-to-github/keeping-your-account-and-data-secure/creating-a-personal-access-token
3. https://stackoverflow.com/questions/15331166/how-to-make-egit-remember-password-and-username#:%7E:text=Go%20to%20the%20Git%20Perspective,username%20and%20password%20%2D%3E%20Select%20Ok

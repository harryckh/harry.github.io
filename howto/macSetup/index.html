The following are some suggested software to be installed in mac.

  1. OpenVanilla: Simplex input that is the same with Window
  2. Rectangle: shortcut for moving window (e.g., cmd + opt + f = full screen) (based on Spectacle) here
  3. Amphetamine: avoid system sleep
  4. Keyboard Maestro: develop Marcos of keyboard/mouse inputs
  5. MonitorControl: Controls your external display brightness and volume and shows native OSD. Use menulet sliders or the keyboard, including native Apple keys! here
  6. Flycut: Clipboard manager
  7. MacDown: an open source Markdown editor for macOS here
  8. TinkerTool: an application that gives you access to additional preference settings Apple has built into macOS. This allows to activate hidden features in the operating system and in some of the applications delivered with the system. here
  9. shottr: screenshot tool + edit here

The following are some suggested setting in mac.

  1. Turn on three-finger drag for your Mac trackpad
    1. Choose Apple menu > System Preferences, then click Accessibility.
    2. Select Pointer Control in the sidebar. ...
    3. Click the Trackpad Options button.
    4. Select ”Enable dragging”, then choose ”three-finger drag” from the menu.
    5. Click OK.
  2. Increase the size of items in sidebars: Choose Apple menu > System Preferences, click General , click the “Sidebar icon size” pop-up menu, then choose Large. The icons and text in the Mail and Finder sidebars, for example, appear larger.
  3. Create two (or more) Chrome in mac:
    1. Download the file: chrome.command e.g., in Desktop
    2. Change username accordingly
    3. run command chmod 775 chrome.command
    4. double click to open it
